Создаем новую папку для нашего проекта и открываем ее через терминал или командную строку. Затем выполняем команду:
```
npx create-react-app football-club-info
```

После завершения установки, переходим в нашу папку:
```
cd football-club-info
```

Теперь можем запустить приложение:
```
npm start
```
